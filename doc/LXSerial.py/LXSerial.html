set_baud_rate ( self,  baud )

Set Baud Rate n, where n is an ASCII digit (1..9) with the following interpertation 1 56.7K 2 38.4K 3 28.8K 4 19.2K 5 14.4K 6 9600 7 4800 8 2400 9 1200 Returns: 1 At the current baud rate and then changes to the new rate for further communication
